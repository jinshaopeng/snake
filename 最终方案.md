# 🎮 贪吃蛇 - 获取APK的最简单方法！

## ⚡ 最快方案：5分钟获取APK（无需安装任何东西）

### 第1步：打开这个网站
👉 **https://webviewgold.com/**

### 第2步：上传文件
- 点击上传，选择本目录下的 `index.html` 文件

### 第3步：填写信息
- App Name: `贪吃蛇`
- Package Name: `com.example.snake`
- 其他默认即可

### 第4步：下载APK！
点击生成，几分钟后就能下载APK了！

| 文件 | 用途 |

## 📱 方案二：使用GitHub Actions自动构建

1. 在 GitHub 创建新仓库
2. 把所有文件推上去
3. 点开 Actions → Build Snake Game APK → Run workflow
4. 等待5-10分钟，下载 Artifacts 里的APK

---

## 💻 方案三：用Android Studio打开项目

1. 下载安装 Android Studio
2. 打开 `android-project/` 目录
3. 等待Gradle同步完成
4. 菜单 Build → Build APK(s)

---

## ✅ 完成

拿到APK后，传到手机安装就可以玩了！
